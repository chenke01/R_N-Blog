# 个人博客系统 - 项目文档

## 项目简介

这是一个基于纯前端技术的个人博客系统，专为GitHub Pages设计。支持Markdown文件直接渲染、标签分类、多主题切换等功能。

## 特性

- ✅ **纯前端实现** - 无需后端服务器，直接部署到GitHub Pages
- ✅ **Markdown支持** - 丢入.md文件即可显示
- ✅ **标签分类** - 自动提取标签，支持分类筛选
- ✅ **多主题切换** - 内置3种主题（暗黑/白色/赛博朋克），支持扩展
- ✅ **科技感设计** - 现代化UI设计，粒子动画效果
- ✅ **响应式布局** - 适配各种屏幕尺寸
- ✅ **代码高亮** - 支持多种编程语言
- ✅ **SEO友好** - 语义化HTML结构

## 技术栈

| 技术 | 用途 | 版本 |
|------|------|------|
| HTML5 | 页面结构 | - |
| CSS3 | 样式和动画 | - |
| JavaScript | 交互逻辑 | ES6+ |
| marked.js | Markdown解析 | 本地加载 |
| highlight.js | 代码高亮 | 11.x（本地加载） |

## 核心模块

### 1. ThemeManager (theme.js)

主题管理系统，负责主题切换和CSS变量管理。

#### 主要功能

- 主题初始化和切换
- CSS变量动态更新
- 主题持久化（localStorage）
- 系统主题跟随
- 主题扩展接口

#### 内置主题

| 主题名称 | 标识 | 说明 |
|----------|------|------|
| 暗黑模式 | dark | 默认主题，深色背景 |
| 白色模式 | light | 浅色背景，护眼 |
| 赛博朋克 | cyberpunk | 紫色霓虹风格，深蓝背景 |

#### 扩展接口

```javascript
// 注册新主题
ThemeManager.registerTheme('custom', {
    name: '自定义主题',
    icon: '🎨',
    colors: {
        '--bg-primary': '#your-color',
        // ... 其他颜色变量
    }
});
```

### 2. MarkdownParser (markdown.js)

Markdown解析器，负责解析frontmatter和渲染Markdown内容。

#### 主要功能

- Frontmatter解析（YAML格式）
- Markdown渲染（使用marked.js）
- 代码高亮（使用highlight.js）
- 摘要提取
- 图片提取

#### Frontmatter格式

```yaml
---
title: 文章标题
date: 2024-01-15
tags: [标签1, 标签2]
author: 作者名
excerpt: 文章摘要
image: 图片URL
---
```

### 3. BlogApp (app.js)

核心应用逻辑，负责文章加载、渲染和交互。

#### 主要功能

- 文章加载和管理
- 标签筛选
- 搜索功能
- 分页显示
- URL路由管理

#### 配置选项

```javascript
config: {
    postsDirectory: 'posts',    // 文章目录
    postsPerPage: 10            // 每页显示数量
}
```

## 页面说明

### 首页 (index.html)

首页包含以下区域：

1. **英雄区域** - 博客标题和描述
2. **统计卡片** - 文章数、标签数、总字数
3. **搜索栏** - 支持标题、内容、标签搜索
4. **标签筛选** - 按标签分类显示文章
5. **文章瀑布流** - 卡片式文章展示
6. **分页** - 文章分页导航
7. **主题切换** - 右下角浮动按钮

### 文章详情页 (article.html)

文章详情页包含：

1. **返回按钮** - 返回首页
2. **文章头部** - 标题、作者、日期、标签
3. **目录** - 自动生成文章目录
4. **文章内容** - Markdown渲染后的内容
5. **文章导航** - 上一篇/下一篇

## 性能优化

1. **本地加载** - 第三方库已下载到本地，无CDN依赖
2. **图片懒加载** - 支持图片懒加载
3. **CSS动画** - 使用CSS3动画，性能更好
4. **系统字体** - 使用系统默认字体，无外部字体依赖

## 安全考虑

1. **XSS防护** - Markdown渲染时进行HTML转义
2. **CSP策略** - 建议配置Content Security Policy
3. **HTTPS** - GitHub Pages默认支持HTTPS

## 已知限制

1. **纯前端限制** - 无法实现真正的后端功能
2. **文件数量** - 大量文章可能影响加载速度
3. **搜索功能** - 基础的客户端搜索
4. **评论系统** - 需要第三方服务
